# Feedback Plan

## Objetivo de la primera semana

Conseguir 20 a 50 jugadores reales que prueben SFL Helper y digan:

- Si el Farm ID carga bien.
- Si el Market les sirve.
- Si Comunidad les parece util para limpiezas.
- Que bug les molesta mas.

## Donde publicarlo primero

1. Amigos o jugadores conocidos de Sunflower Land.
2. Discord oficial, si las reglas permiten compartir herramientas no oficiales.
3. Telegram oficial o grupos de comunidad.
4. X / Twitter con capturas.
5. Reddit o foros de Web3 gaming, si encuentras comunidades activas.

## Como pedir feedback sin sonar spam

Usa este formato:

Hola, estoy probando una herramienta beta para Sunflower Land. No vendo nada ni pide wallet, solo Farm ID y datos publicos. Me serviria mucho que la prueben y me digan que falla o que falta.

Link: https://sfl-helper.vercel.app/

## Preguntas para usuarios

1. Tu Farm ID cargo bien?
2. Los precios se ven correctos comparados con el juego?
3. Publicarias tu granja para pedir limpiezas?
4. Que parte se entiende menos?
5. Que agregarias antes del lanzamiento 0.2?

## Respuestas rapidas para dudas

Pregunta: Es oficial?
Respuesta: No, es una herramienta no oficial hecha por la comunidad.

Pregunta: Pide wallet?
Respuesta: No. Solo usa Farm ID y datos publicos.

Pregunta: Por que mi dato no aparece actualizado?
Respuesta: La app depende de datos publicos de SFL World/Sunflower Land, que pueden tener retraso.

Pregunta: Para que sirve Comunidad?
Respuesta: Para publicar tu granja y que otros jugadores puedan visitarte y ayudarte con limpiezas.

## Prioridad de bugs

Alta:
- La app no carga.
- No se puede publicar en Comunidad.
- El Market muestra precios claramente malos.
- El Farm ID no sincroniza.

Media:
- Problemas visuales en celular.
- Traducciones raras.
- Links de market incorrectos.

Baja:
- Mejoras esteticas.
- Nuevas funciones.
- Cambios de colores o layout.
